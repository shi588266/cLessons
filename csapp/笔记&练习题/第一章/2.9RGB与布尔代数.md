### 2.9 Rgb与布尔代数
- 基于光源R红G绿B蓝的关闭(0)或打开(1), 我们能创建 2*2*2=8种不同的颜色   

    | R | G | B | color |
    |:------:|:------:|:---------:|:---------:|
    | 0 | 0 | 0 | 黑色 |
    | 0 | 0 | 1 | 蓝色 |
    | 0 | 1 | 0 | 绿色 |
    | 0 | 1 | 1 | 蓝绿色 |
    | 1 | 0 | 0 | 红色 |
    | 1 | 0 | 1 | 红紫色 |
    | 1 | 1 | 0 | 黄色 |
    | 1 | 1 | 1 | 白色 |

    __Question A__: 每种颜色的补  

    | R | G | B | color |
    |:------:|:------:|:---------:|:---------:|
    | 1 | 1 | 1 | 白色 |
    | 1 | 1 | 0 | 黄色 |
    | 1 | 0 | 1 | 红紫色 |
    | 1 | 0 | 0 | 红色 |
    | 0 | 1 | 1 | 蓝绿色 |
    | 0 | 1 | 0 | 绿色 |
    | 0 | 0 | 1 | 蓝色 |
    | 0 | 0 | 0 | 黑色 |

    __Question B__: 描述下列颜色应用布尔运算的结果  

    - 蓝色 | 绿色 = 001 | 010 = 011(蓝绿色)
    - 黄色 & 蓝绿色 = 110 & 011 = 010(绿色)
    - 红色 ^ 红紫色 = 100 ^ 101 = 001(蓝色)
